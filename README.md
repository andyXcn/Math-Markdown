**Was ist es und wofür ist es gut?**\
Erlaubt es, leichten Markdown Text zu verwenden mit einbindung mathematischer Formeln. Die Eingabe erfolgt links, während rechts live kompiliert wird.\
\
**Commands:**\
"#" für Überschrift 1\
"##" für Überschrift 2\
...\
"* Text *" (**Fett**)\
"_ Text _" (*kursiv*)\
" __ Text __" (unterstrichen)\
" ~ Text ~ " (~~durchgestrichen~~)\
\
"-" bzw. " * " für ungeordnete Liste\
"1." für numerierte Liste (beliebiger Start)\
\
Inline Math: "$Formel$"\
Block Math: "$$Formel$$"
